Background

I joined Huawei as a data analyst intern in the summer of 2021, amidst the COVID-19 pandemic. Therefore, I complemeted the entire work term remotely. I worked with data analysts and software engineers to develop an efficient pipeline for processing numerical and textual data.

Contributions

Building Web Application with Flask for Data Processing

The Flask app provided an internal interface for data analysts to call data processing scripts. Prior to building the app, each data analyst needed to pull scripts from the repository and read through manuals before executing each. The app eliminated user effort by simply asking the user to upload input files and choose the scripts to run. It also enabled users with no technical backgrounds to receive processed data.

I assisted in building the display of the interface with HTML/CSS, and connected user input files to the target scripts to run. I also developed one of the data-processing scripts used in the backend of the app. The interface reduced overall processing time by 80%.

Methods for Determining Semantic Textual Similarity

One of my tasks was to process the semantics behind text data. Examples include highlighting synonyms of a target word in another piece of text and finding keywords to summarize a paragraph.

I adopted concepts from machine learning and translated text data to numerical data through embeddings. I implemented methods to evaluate the different distances between words, reflecting their similarity score.
